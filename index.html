QxMaven

Web development with Qooxdoo, powered by Maven

Overview

QxMaven aims to provide a complete solution for developing web applications by using the Qooxdoo Javascript framework and Maven.

The project is composed of several maven artifacts:

  • qooxdoo-maven-sdk provides a single package which contains the entire framework, including the web tools to help in development (API viewer, unit test, inspector, â€¦)
  • qooxdoo-maven-plugin allows maven to compile, optimize, and test your qooxdoo application
  • qooxdoo-archetype-desktop provides a skeleton project for quick starting the development of a qooxdoo desktop application
